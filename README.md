# Task Tracker

## Overview
A simple Task Tracker web application built using HTML, CSS, JavaScript, and TypeScript.

## Features
- Add tasks
- Mark tasks as Done/Undo
- Filter tasks (All, Pending, Done)
- Sort by Priority and Due Date
- Clear all tasks
- Save tasks using Local Storage

## Technologies Used
- HTML
- CSS
- JavaScript
- TypeScript

## How to Run
1. Clone the repository.
2. Run `npx tsc` to compile TypeScript.
3. Open `index.html` using Live Server.
